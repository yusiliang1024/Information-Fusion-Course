# 场景：自动驾驶汽车的障碍物测距
假设你的车正在向前行驶，前方有一辆静止的卡车。你车上装有两个传感器：

- 激光雷达 (LiDAR)：精度高，但在大雾天表现稍差。

- 毫米波雷达 (Radar)：精度略低，但穿透力强，不受天气影响。

1. 数据采集与 L1 目标评估激光雷达观测值 ($z_1$)：测量距离为 10.2 米。根据规格，它的方差（不确定性）$\sigma_1^2 = 0.01$。
2. 毫米波雷达观测值 ($z_2$)：测量距离为 9.7 米。它的方差 $\sigma_2^2 = 0.09$（意味着它没那么准）。
  - $\sigma_2^2 $可以通过$$\sigma^2 = \frac{1}{n-1} \sum_{i=1}^{n} (z_i - \bar{z})^2$$测量得到，
    $z_i$ 是第 $i$ 次测量的结果，$\bar{z}$ 是这 1000 次结果的平均值
## 融合处理（贝叶斯/加权平均）
我们不能简单地取算术平均值 $(10.2 + 9.7) / 2 = 9.95$。
科学的做法是根据不确定性分配权重：$$W_1 = \frac{\sigma_2^2}{\sigma_1^2 + \sigma_2^2} =\frac{0.09}{0.01 + 0.09} = 0.9$$
$$W_2 = \frac{\sigma_1^2}{\sigma_1^2 + \sigma_2^2} = \frac{0.01}{0.01 + 0.09} = 0.1$$
- 融合结果：$10.2 \times 0.9 + 9.7 \times 0.1 = \mathbf{10.15 \text{ 米}}$。
- 结论：因为激光雷达更准，融合结果更偏向于它。融合后的总方差减小到了 $0.009$，比任何一个单一传感器都更可靠
- 假设我们有两个传感器对同一目标的测量值 $z_1$ 和 $z_2$，它们的方差分别是 $\sigma_1^2$ 和 $\sigma_2^2$。为了得到最精确的估计，我们希望融合后的方差 $\sigma_{fused}^2$ 达到最小。经过数学推导（拉格朗日乘数法或最小方差估计），你会得到这个经典的计算公式：$$\frac{1}{\sigma_{fused}^2} = \frac{1}{\sigma_1^2} + \frac{1}{\sigma_2^2}$$或者
  变换成我们常用的形式：$$\sigma_{fused}^2 = \frac{\sigma_1^2 \cdot \sigma_2^2}{\sigma_1^2 + \sigma_2^2}$$
