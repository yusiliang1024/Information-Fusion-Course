# 问题的表达
 1. 假设检验问题
    给定一组假设 $H_i$, 通过对已有的数据集 $y$ 进行处理，确定当前哪一个假设 $H_i$ 成立， 从而做出决策 $D_j$.
    - 二元
      只有  $(H_0, H_1)$ 的假设。
      - 例如：雷达判断监视区内是否存在目标
        $H_0$ :有目标， $H_1$ :无目标
      - 二元问题假设建立
      - 虚警和漏检
        二元假设中： 检测率 = 1 - 漏检率
    - 多假设（最优假设问题）
      根据已有数据集 $y$ 得到最优决策结果 $D_x$
      
     
# 分布式并行检测融合策略（算法）
  1. 概念：每个局部传感器基于自己的观测值 $y_i$ 完成同一个决策任务。之后将决策值 $u_i$ 传输到融合中心，做出决策。利用**最优融合规则**，做出最终决策 $u_0$
  2. 融合中心的决策值，准则： 对于二元问题，有四种可能性。
     - $H_0$ 为真， $u_0=0$ 。
     - $H_1$ 为真， $u_0=1$ 。
     - $H_0$ 为真， $ u_0=1 $  。
     - $H_1$ 为真， $ u_0=0 $  。
     - 虚警率: $P_f=P\left( u=1\left| H_0 \right. \right) $
     - 漏检率：$P_m=P\left( u=0\left| H_1 \right. \right) $
     - 检测率：$P_d=P\left( u=1\left| H_1 \right. \right) $
     -  对于检测率有公式： $P_d=1-P_m$
     - $R_0$ : 接受区，认为假设 $H_0$ 而成立做出的决策 $D_0$ 。
     - $R_1$ : 拒绝区，拒绝 $H_0$ ，认为假设 $H_1$ 而成立做出的决策 $D_1$ 。
     > 两个区域要包含空间所有的点
     - 一般二元假设检验问题描述：假设 $p_0(y), p_1(y)$ 分别为 $y$ 关于 $H_0$ 和 $H_1$ 的条件概率密度，则虚警和漏检概率分别为，虚警概率： $P(D_1|H_0)=\int_{R_1}{p_0\left( y \right) dy}$漏检概率： $P(D_0|H_1)=\int_{R_0}{p_1\left( y \right) dy}$

     - 在分布式多传感器检测系统中，如何根据各个传感器的决策得到融合中心的决策值，有多种准则。假设，系统中有N个传感器，其中第i个传感器的检测率为 $P_d^i$ , $P_f^i$ , $P_m^i$ 。 $u_i$ 表示第 $i$ 个传感器的决策值， $u_i$ 表示第 $i$ 个传感器的决策值 $u_0$ 为按照融合策略得到的全局最优决策，以下为相关决策： 
        1. **“与”融合检测准则**：可以减少虚警率，检测概率降低。
        2. **“或”的准则**： 可以提高检测概率，虚警概率提高。
           $P_d=1-\prod_{i=1}^N{\left( 1-P_{d}^{i} \right)}, P_f=1-\prod_{i=1}^N{\left( 1-P_{f}^{i} \right)}$
        4. **表决融合检验准则**： $u_0=\begin{cases} 1,\sum{u_i}\geqslant k\\	0, \sum{u_i}<k\\ \end{cases}$   
          ![df061e63d6d439c2799167a35308e7e1](https://github.com/user-attachments/assets/8f7b9ccc-cbc5-4234-aeed-46ced43be298)
          - *k准则融合判则* ：k取值比较关键，可以满足一定虚警率前提下提高检测率，或两者进行权衡，根据实际要求。
        > **不足**：以上检测准则都没有考虑各个假设出现的概率，及各个传感器的概率特性，因此虽然简单，但不适用。
          - 例如，教室里出现老虎和小狗的概率不同，以上这三个准则无法体现。
        4. **最大后验概率融合检测准则**：给定全局观测u的前提下，$H_j$ 为真的概率。融合规则： $\frac{P\left( H_1|u \right)}{P\left( H_0|u \right)}>1?H_1:H_0$ 应用贝叶斯法则： $P\left( H_i|u \right) =\frac{P\left( u|H_i \right) P\left( H_i \right)}{P\left( u \right)}$ 。 所以准则也可以写为:  $\frac{P\left( u|H_1 \right)}{P\left( u|H_0 \right)}>\frac{P\left( H_0 \right)}{P\left( H_1 \right)}, P\left( H_0 \right)\text{和}P\left(H_1\right)\text{为先验概率}$ 也可以写为 $\lambda \left( u \right) >\lambda _0?H_1:H_0$ 。 $P\left( u|H_1 \right)$ 和 $P\left( u|H_0 \right)$ 为似然函数，也称似然比检验。
        6. **Neyman-Pearson融合准测**：控制虚警率
