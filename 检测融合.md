# 问题的表达
 1. 假设检验问题
    给定一组假设 $H_i$, 通过对已有的数据集 $y$ 进行处理，确定当前哪一个假设 $H_i$ 成立， 从而做出决策 $D_j$.
    - 二元
      只有  $(H_0, H_1)$ 的假设。
      - 例如：雷达判断监视区内是否存在目标
        $H_0$ :有目标， $H_1$ :无目标
      - 二元问题假设建立
      - 虚警和漏检
        二元假设中： 检测率 = 1 - 漏检率
    - 多假设（最优假设问题）
      根据已有数据集 $y$ 得到最优决策结果 $D_x$
      
     
# 分布式并行检测融合策略（算法）
  1. 概念：每个局部传感器基于自己的观测值 $y_i$ 完成同一个决策任务。之后将决策值 $u_i$ 传输到融合中心，做出决策。利用**最优融合规则**，做出最终决策 $u_0$
  2. 融合中心的决策值，准则：
     对于二元问题，有四种可能性。
     - $H_0$ 为真，$u_0=0$ 。
     - $H_1$ 为真，$u_0=1$ 。
     - $H_0$ 为真，$u_0=1$ 。
     - $H_1$ 为真，$u_0=0$ 。
       虚警率: $P_f=P\left( u=1\left| H_0 \right. \right) $
       漏检率：$P_m=P\left( u=0\left| H_1 \right. \right) $
       检测率：$P_d=P\left( u=1\left| H_1 \right. \right) $
       $R_0$ : 接受区，认为假设 $H_0$ 而成立做出的决策 $D_0$ 。
       $R_1$ : 拒绝区，拒绝 $H_0$ ，认为假设 $H_1$ 而成立做出的决策 $D_1$ 。
       > 两个区域要包含空间所有的点
       
      1. **“与”融合检测准则**：可以减少虚警率，检测概率降低。
      2. **“或”的准则**： 可以减少检测概率，虚警概率提高。
      3. **表决融合检验准则**：
        - *k准则融合判则* ：k取值比较关键，可以满足一定虚警率前提下提高检测率，或两者进行权衡，根据实际要求。

        - 例如，教室里出现老虎和小狗的概率不同，以上这三个准则无法体现。
